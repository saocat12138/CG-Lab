## 四、实现原理

### 1. 贝塞尔曲线 — De Casteljau 算法
给定控制点序列 `P₀, P₁, ..., Pₙ₋₁`，参数 `t ∈ [0,1]`：
1. 对相邻控制点做线性插值：`P' = (1-t)Pᵢ + tPᵢ₊₁`
2. 递归插值，直到只剩**一个点**，即为曲线上的点
3. 遍历 t=0 → 1，采样得到完整曲线

### 2. 光栅化渲染
1. 建立 800×800 像素缓冲区 `pixels`
2. 将归一化坐标 `(x,y) ∈ [0,1]` 映射为屏幕像素坐标
3. 直接修改像素 RGB 值完成绘制
4. **批量上传数据**：CPU 计算全部点 → 一次性传入 GPU → 并行绘制

### 3. 反走样（抗锯齿）
- 对浮点坐标点周围 **3×3 像素邻域** 计算距离
- 使用**距离衰减权重**：点越近，亮度越高
- 多像素混合，消除阶梯状锯齿，实现平滑边缘

### 4. 均匀三次 B 样条曲线
- 每 **4 个控制点** 生成一段曲线
- n 个控制点生成 **n-3 段**，自动 **C2 连续平滑拼接**
- **局部控制**：修改一个控制点只影响附近曲线
- 采用固定三次 B 样条基矩阵计算，稳定高效

## 六、核心代码模块说明

### 1. De Casteljau 算法（贝塞尔核心）
```python
def de_casteljau(points, t):
    if len(points) == 1:
        return points[0]
    next_points = []
    for i in range(len(points)-1):
        p0, p1 = points[i], points[i+1]
        x = (1-t)*p0[0] + t*p1[0]
        y = (1-t)*p0[1] + t*p1[1]
        next_points.append([x, y])
    return de_casteljau(next_points, t)
```

@ti.kernel
def draw_antialiased_curve(n: ti.i32):
    for i in range(n):
        x, y = curve_points_field[i]
        fx = x * WIDTH
        fy = y * HEIGHT
        for dx, dy in ti.static(ti.ndrange(-1,2,-1,2)):
            cx = ti.cast(fx + dx, ti.i32)
            cy = ti.cast(fy + dy, ti.i32)
            if 0<=cx<WIDTH and 0<=cy<HEIGHT:
                dist_sq = (fx-(cx+0.5))**2 + (fy-(cy+0.5))**2
                weight = ti.exp(-dist_sq * 2.0)
                pixels[cx, cy] += ti.Vector([0.0, weight, 0.0])

## 七、实验现象与对比

### 1. 贝塞尔曲线
- 控制点 ≥2 即可绘制
- **全局控制**：移动任意点，整条曲线变形
- 控制点越多，曲线阶数越高，易波动

### 2. B 样条曲线
- 控制点 ≥4 才显示
- **局部控制**：移动一个点，仅影响局部
- 曲线更平稳、易于编辑
- 天然光滑拼接（C2 连续）

### 3. 反走样效果
- 基础绘制：边缘呈**阶梯状锯齿**
- 反走样后：边缘**平滑柔和**，无锯齿
