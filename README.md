# 实验二：3D空间MVP坐标变换（Taichi实现）

## 一、实验目标
1. 深入理解 3D 空间中的坐标变换流程（模型-视图-投影/MVP变换）。
2. 独立推导并用代码实现模型变换、视图变换和投影变换矩阵。
3. 掌握面向数据编程框架 Taichi 的基本语法与矩阵操作。

## 二、实验背景
本实验基于Taichi框架，对三维空间中的三角形顶点进行MVP变换，将3D坐标转换为2D屏幕坐标，并在GUI窗口中绘制线框三角形。
三角形初始顶点坐标：
- $v_0$: (2.0, 0.0, -2.0)
- $v_1$: (0.0, 2.0, -2.0)
- $v_2$: (-2.0, 0.0, -2.0)

## 三、核心实现（三个关键矩阵）
### 1. 模型变换矩阵（get_model_matrix）
- 功能：接收角度值（角度制），返回绕Z轴旋转的4×4齐次变换矩阵。
- 核心逻辑：角度转弧度→计算余弦/正弦值→构建旋转矩阵。
